## 2026.04 업데이트

### 🗓️ 시간표 기능 전면 개편

#### 새로 추가된 것들

- **교사 시간표 페이지** 추가 (`/timetable/t`)
  - 선생님 이름을 선택하면 해당 선생님의 주간 시간표를 확인할 수 있어요
  - 마지막으로 선택한 선생님이 자동으로 기억됩니다

- **수업 변경 교사 등록** 기능 추가 (관리자)
  - 나이스에서 수업이 바뀐 교시에 담당 교사를 직접 등록할 수 있어요
  - 저장하면 학생 시간표에 즉시 반영됩니다

- **기본 시간표 수정** 기능 추가 (관리자)
  - 나이스에 등록되지 않은 교시(기본 시간표 기준)를 수정할 수 있어요
  - 시험 / 자습 / 수업없음 등으로 변경 가능하고, 직접 입력도 됩니다
  - 초기화하면 기본 시간표로 다시 돌아가요

#### 바뀐 것들

- **학생 시간표** 범례 추가
  - `변경` 뱃지: 나이스 기준으로 수업이 바뀐 교시
  - 회색 점: 기본 시간표 기준 (나이스 미등록) 교시

- **교시별 담당 교사 표시** 개선
  - 해당 교시에 배정된 선생님만 표시되도록 변경 (여러 명 동시 표시되던 문제 수정)
  - 수업이 변경된 경우에는 변경된 수업의 담당 교사만 표시

- **교사 매핑 관리** 개선 (관리자)
  - 반별로 따로 관리하던 방식 → 학년 단위로 통합 관리로 변경
  - 태그 형태로 교사를 추가/삭제할 수 있게 변경

- **기본 시간표 관리** 개선 (관리자)
  - 교시별 담당 교사 설정 기능 추가
  - 해당 과목에 선생님이 2명 이상인 경우에만 선택 드롭다운이 표시됩니다
